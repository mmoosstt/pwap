# pwap
## Progressive Web Application Playground / PWA as executable
This is a project to get in touch with progressiv web applications from a Python and VUE point of view.
![Context](overview.svg)

## Setup Python venv

```
    py -3 -m pip poetry
    py -3 -m venv .venv
    py -3 -m pip poetry
    py -3 -m poetry update
```
